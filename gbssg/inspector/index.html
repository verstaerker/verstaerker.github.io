Excersises

NOTE: reloading the page in between changes will reset the page!

  1. What's the name of the font used for the h3 titles?
  2. What color value has the "Submit" buttons background when inactive?
  3. What color value is used for the "Submit" buttons background when it's hovered?
  4. Change the background-color of the "Submit" button in default and hover mode by using the color picker of the inspector.
  5. Inspect the layout changes between the desktop and mobile view. Instead of resizing the browser use the "device" or "responsive" mode.
  6. What could be improved on the mobile layout?
  7. Improve the style for the subtitles so the sections become better distinguishable.

    You could use the following properties to do so:
    • margin: outer spacing - you can define a single value for all directions or one for each direction: [top] [right] [bottom] [left]
    • padding: inner spacing- you can define a single value for all directions or one for each direction: [top] [right] [bottom] [left]
    • color: font color (e.g. blue)
    • border - define value as [width (e.g. 1px)], [style (e.g. solid)], [color (e.g. red)]
    You can find additional style options on https://cssreference.io/ (e.g. search for font).
  8. The developer tools of Chrome and Firefox allow to inspect the animation on the submit button. Did you find it?
